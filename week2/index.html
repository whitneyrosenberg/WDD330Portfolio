JavaScript: Novice to Ninja - Chapter 2

  • The syntax used by JavaScript is known as a C-style syntax because of its similarities with the C programming language.
  • Reserved Words:
  • Important, but not reserved: undefined, NaN, Infinity
  • Prmitive Data Types:
    • String
    • Symbol (introdcued in ES6)
    • Number
    • Boolean
    • Undefined
    • Null
  • If the value isn't a primitive data type then it is an object
  • Use const and let rather than var when declaring variables. This ensures variables are block scoped, meaning their value only exists inside the block they are declared in.
  • Variable names that start with an underscore generally refer to private properties and methods
  • The $ character is also used by the jQuery library.
  • If you assign a non-primitive value to a variable, then this is done by reference, so any changes that are subsequently made will affect all references to that object.
  • Use template literals (Strings with back ticks rather than quotation marks) to allow interpolation of JS code in a string.
    				const name = `Whitney`;
`Hello! My name is ${ name }!`;
<< 'Hello! My name is Whitney!';

JavaScript: Novice to Ninja - Chapter 3

Arrays
  • We can use index notation to add new itesm to any position in an array. Positions that are left empty will be undefined.
  • Use the delete operator to remove an item from an array. However, this does not alter the number of elements in the array. The space where the value was removed will remain, but now with the value of undefined. 
    				let arr = [1, 2, 3];
delete arr[2];
console.log(arr);
<< [1, 2, undefined]
  • Use the pop() method to remove the last item from an array.
  • Use the shift() method or removed the first item from an array.
  • Use the push() method to append a new item to the end of an array.
  • Use the unshift() method to append a new item to the beginning of the array.
  • Use the concat() method to merge two arrays.
  • Use the join() method to turn the array into a string that comprises of all the items in the array separated by commas. You can choose a separator other than the comma by placing it inside the parantheses.
  • Use the slice() method to create a sub array by passing in the first and last indices containing the values for the new array. This is a non-destructive operation.
  • The splice() method removed items from an array then inserts new items in their place by passing in the index of the first value to be removed, the number of values to remove, and the new values to be added. This is a destructive operation.
  • Use the reverse() method to reverse the order of an array. This changes the order of the array permanently.
Sets
  • A set is a data structure that represents a colletion of unique values.
  • They are a useful way to keep track of data without having to check for duplicates.
  • Create a set using the Set() constructor.
  • Add items to a set using the add() method. If the set already containts the value, it will be ignored.
  • Check the size of a set with the size() method.
  • Use the has() method to check if a value is in a set.
  • Use the delete() method to remove a value from a set.
  • Use the clear() method to remove all values from a set.
  • Convert a set to an array using the spread operator in an array literal or using Aray.from().
Maps
  • Maps are data structures that contain key and value pairs.
  • Create a map using the Map() constructor.
  • Add entries to maps using the set() method with the key and value as the arguments.
  • Use the get() method by passing in a key to get the associated value.
  • Use the has() method by passing in a key to check if that particular key in the map.
  • Use the size property to get the number of key values pairs in a map.
  • Use the delete() by passing in a key to remove a value from a map.
  • Use the clear() method to remove all key and value pairs from a map.
  • Convert a map to an array using the spread operator in an array literal or using Aray.from().

JavaScript: Novice to Ninja - Chapter 4

  • There are a number of ways to define a function in JavaScript. These are the 3 most common:
    1. function declaration will give a "named function". 
      function hello() { }
    2. Function expressions can create both named and anonymous functions. 
      const hello = function() { }
      or 
      const hello = function sayHello() { }
    3. Function() constructor
      const hello = new Function('console.log("Hello World!");');
  • Keep your code DRY - Don't Repeat Yourself!
    • This can be done using functions for code that needs to be repeated.
  • You can declare a function that accepts a variable number of arguments by using rest operator by placing three dots in front of the last parameter in a function declaration. This will then collect all the arguments entered into an array.
  • You can specify default parameters by assigning the defalt value to the parameter in the function definition.
  • Declaring functions as arrow functions can be useful for short anonymous functions by using less verbose syntax. 
    const hello = () => alert('Hello World!');
Callbacks
  • Functions can be passed as arguments for other functions. These are called callback functions.
  • The function name is passed in as an argument, then the function can be invoked with parentheses.
